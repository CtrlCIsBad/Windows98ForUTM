# Windows98ForUTM
This repo has windows 98 pre-configured and pre-installed .utm files.

Pre-installed just means that you download the .utm file, then open it in UTM/UTM SE, and then you can easily get windows 98 in UTM/UTM SE. Basically you don't need to have an .iso and don't need to install it (which takes time.) If you don't want to take your time in setup, pre-installed is for you.


Pre-configured is for those who prefer to get their own .iso, and then take their time in setup. You can customize anything in setup as you like.

If you are going with pre-installed, just scroll down below pre-configured.

If you are going with pre-cofigured, you need to obtain your .iso file. An .iso I recommend is https://archive.org/details/windows-98-se-isofile

It has an valid product key, which is legal.
